Piston Test Page

The MolecularPiston Physlet is part of the molecular dynamics package.  The movable piston in the above simulation acts upon an ensemble in one of two ways.   In "parse mode" the piston is scripted to follow a fixed path, h(t).   If "parse mode" is set to false, the piston is a freely falling object that is acted on by a constant downward force.  Both the piston mass and the force can be set when the applet is embedded.
