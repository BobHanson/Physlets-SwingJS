Diatomic models a mixture of monatomic and diatomic molecules.   The Diatomic Physlet is designed to be scripted using a scripting language such as JavaScript.  This feature requires a Java 1.1 capable browser with JavaScript to Java scripting capability.  Please see the demo pages below. 

Diatomic was written by Ernesto Martin and Miguel Celdran at the Universidad de Murcia, Spain using Easy Java Simulations. It was modified by  Wolfgang Christian to include inter-applet communication.
