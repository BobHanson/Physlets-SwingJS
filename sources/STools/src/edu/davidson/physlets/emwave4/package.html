EMWave allows users to show the relationship between electric and magnetic fields in a traveling E&M wave.

Although some Physlets can be used in interactive mode, EMWave must be scripted using a scripting language such as JavaScript and requires a Java 1.1 capable browser with JavaScript to Java scripting capability.
