MolecularApplet Embedding

Molecular contains four kinetic theory applets based on a hard disk model.  The basic applet, MolecularApplet, models a gas using either periodic or hard wall boundary conditions.

Molecular is embedded using the following html tag:

<applet
  codebase="./classes/"
  archive="Molecular4_.jar,STools4.jar"
  code="molecular.MolecularApplet.class"
  name="Molecular"
  width="400"
  height="350">

<param name="ShowControls" value="true" />
<param name="InitialP" value="20" />
<param name="FPS" value="10" />
<param name="dt" value="0.1" />
<param name="PeriodicH" value="false" />
<param name="PeriodicV" value="false" />
<param name="Border" value="2" />
<param name="MaxParticles" value="100" />
<param name="PixPerUnit" value="10" />
</applet>
