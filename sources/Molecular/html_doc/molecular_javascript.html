  • Add Particles: Scripts in a few particles with solid boundaries. Particles have different masses.
  • One Particle: Add a particle to the simulation.
  • Assorted Particles  Periodic boundaries in the x direction.

Scripts on this page test the functionality of the basic hard-sphere ensemble from which other, more specialized ensembles, are derived.  The MolecularApplet Physlet is a simple applet that allows scripting.  It is possible to create overlapping particles but this is non-physical since hard spheres cannot penetrate. We ignore interactions between overlapping particles.
