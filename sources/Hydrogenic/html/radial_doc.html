Radial Embedding

Radial shows the radial dependence of the hydrogenic wave function.

Radial is embedded using the following html tag:

<applet
  codebase="../classes/"
  archive="Hydrogenic4_.jar,STools4.jar"
  code="hydrogenic.Radial.class"
  name="radial"
  width="450"
  height="300">

<param name="l" value="0" />
<param name="n" value="1" />
<param name="ShowControls" value="true" />
<param name="MultiPlot" value="false" />
</applet>
