BField simulates magnetic fields from coils and wires.

Coils and wires are added to region that contains a static background field.  The resulting magnetic field

BField can be used in interactive mode.

BField can be scripted using a scripting language such as JavaScript and requires a Java 1.1 capable browser with JavaScript to Java scripting capability.
