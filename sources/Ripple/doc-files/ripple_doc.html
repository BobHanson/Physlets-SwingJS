Ripple Embedding

Ripple models a water-filled tank containing one or more point-source exciters.

Ripple is embedded using the following html tag:

<applet
  code="ripple.Ripple.class"
  codebase="../classes"
  archive="Ripple4_.jar,STools4.jar"
  name="Ripple"
  width="320"
  height="370">

<param name="numberOfSources" value="3" />
<param name="pixPerUnit" value="20" />
<param name="wavelength" value="2.0" />
<param name="showControls" value="true" />
<param name="showIntensity" value="true" />
<param name="animate" value="true" />
<param name="fps" value="10" />
<param name="caption" value="Ripple Tank" />
</applet>
